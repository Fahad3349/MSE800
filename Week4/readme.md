
## Overview
This project is a simple Object-Oriented Python application that calculates
the area and perimeter of a rectangle based on user input.

## Files

### 1. rectangle.py
Contains the Rectangle class with:
- Constructor to initialize dimensions
- Method to calculate area
- Method to calculate perimeter

### 2. main.py
Handles:
- User input
- Object creation
- Displaying results

## Example

Enter the length: 5  
Enter the width: 3  

Area: 15  
Perimeter: 16
